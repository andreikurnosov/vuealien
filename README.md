# Vuealien

Simple game to find out who is AlienZombie!

Deployed to Netlify - [Vuealienzombie](https://vuealienzobmie.netlify.app/).

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
